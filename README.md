# ruby-grid
Ruby library to work with matrices of integers.

## Main Goal
To manipulate (=play_with) images using Ruby.

So far only basic methods and relative overloads are implemented.
Tested.

## Usage
Please refer to the included autogenerated doc.

## Is it any good?
Time will tell.

So far, so fun!
